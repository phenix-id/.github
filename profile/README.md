### Overview
Phenix-ID is dedicated to creating infrastructure that empowers users with sovereign control over their digital identities. We focus on high-security, privacy-respecting tools that bridge the gap between users and digital services.

### Core Services
Our ecosystem is built around four primary pillars designed for secure digital identity management:

* **Agent Controller**: Orchestrates identity agents and manages their lifecycle and interactions.
* **Platform**: The underlying infrastructure and API layer supporting the entire identity network.
* **Studio**: A specialized environment for configuring, managing, and visualizing identity workflows.
* **Mediator**: Facilitates secure, asynchronous communication between identity agents (DIDComm).
